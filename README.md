## 🚀 Overview
**WortMind** is an interactive, flashcard-based learning application designed to help students master German vocabulary. Built with Python, it focuses on bridging the gap between Arabic and German by providing contextual examples and structured vocabulary management.

## ✨ Key Features
* **Dual Language Support:** Accurate translations from German to Arabic.
* **Contextual Learning:** Each word comes with a practical example sentence (*Beispielsatz*) to improve usage understanding.
* **Structured Progress:** Vocabulary is organized into "Boxes" to facilitate spaced repetition and memory retention.
* **CI/CD Integrated:** Automated testing and linting ensure the code remains clean and reliable.

## 🛠️ Technical Stack
* **Language:** Python 3.10+
* **Data Format:** JSON (for easy vocabulary management and portability).
* **DevOps:** GitHub Actions for Continuous Integration (Pytest & Flake8).

## 📂 Project Structure
- `app.py`: The main entry point for the application.
- `data/words.json`: The database of German words and their translations.
- `utils/storage.py`: Logic for saving and retrieving learning progress.
- `tests/`: Automated unit tests to ensure application stability.
