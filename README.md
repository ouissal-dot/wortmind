# 🧠 WortMind – Learn German Vocabulary

🚀 Live Demo: https://wortmind-evu4aw4nknppht39m36rdr.streamlit.app/

WortMind is a simple and interactive flashcard app built with Streamlit to help users learn German vocabulary with Arabic translations.

---

## ✨ Features

* 📦 Organize words into learning boxes
* 🎴 Flashcard-style learning (German → Arabic)
* 🔁 Flip cards to reveal meanings
* ➡️ Random word practice
* ➕ Add your own vocabulary
* 💾 Persistent storage (load/save data)
